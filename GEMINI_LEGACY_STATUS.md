# Gemini CLI Legacy Status

## Compatibility Preservation Strategy
Everything Gemini Code (EGC) remains committed to preserving the stability of the Gemini CLI ecosystem. This repository's current state represents the pinnacle of Gemini-compatible engineering. We will continue to maintain this branch for as long as it remains operationally viable.

## Legacy Support Policy
- **Security:** Critical security patches will be prioritized for the legacy branch.
- **Documentation:** All existing Gemini-specific documentation will be preserved and archived.
- **Transition Assistance:** We will provide clear migration paths for users moving to newer runtimes.

## Accessing the Legacy Version
Users requiring a stable, unmodified Gemini CLI environment should utilize the following Git tags:
- `legacy/v1.10.0`
- `compatibility/gemini-cli-final`
