# EGC - Дайте Каждому ИИ-Агенту Один Общий Мозг

**Постоянная память, которую каждый ИИ-агент, IDE, терминал и сессия разделяют автоматически. Не нужно запоминать промпты. Не нужно восстанавливать контекст. Просто говорите.**

</div>

---

EGC не очередной инструмент памяти. Это слой интеллекта, благодаря которому любой ИИ работает так, будто он в вашем проекте с первого дня, в Cursor, Copilot, Claude Code, Codex, Aider и любом терминальном агенте (всего 20 ИИ-инструментов для программирования). Нативно работает с Claude, GPT-4o, Gemini, DeepSeek, Mistral, Groq, Cohere и Vertex AI, плюс OpenRouter для Qwen3, Llama 4 и других.

Каждый разговор наращивает коллективный интеллект вашего проекта. Каждый агент его наследует. Каждая сессия становится умнее.

## Установка

```bash
npm install -g @egchq/egc && egc install
```

- **Сократите потери контекста до 90%, уменьшите расходы на токены и держите все ИИ идеально синхронизированными между сессиями.**
- **Guardian: проверяет каждую команду перед выполнением, блокирует опасные записи и обнаруживает prompt injection. Каждый общий мозг поставляется со встроенным слоем безопасности.**
- **Одна команда, ноль настроек: память хранится локально и зашифрованно на вашей машине и никогда не коммитится в git.**

## Внутри Мозга: Как Работает EGC

EGC не список инструментов; это один мозг с несколькими способностями. Он помнит, понимает, защищает, фильтрует и координирует, во всех ИИ-агентах на вашей машине.

<div align="center">
  <img src="../../assets/sharedbrain.gif" alt="Cursor to Claude Code shared memory" width="900" />
</div>

### Вы Не Запоминаете Команды, Вы Просто Говорите

Говорите с мозгом на любом языке: «сохрани эту сессию», «что мы решили по авторизации?», «запомни это решение». EGC понимает намерение, сохраняет контекст и мгновенно восстанавливает его в любой другой вкладке, терминале или инструменте на вашей машине. Один мозг. Все агенты. Ноль команд для запоминания.

### Постоянная Память Проекта

EGC даёт каждому ИИ-агенту постоянный общий мозг. Он фиксирует решения, контекст сессии, рабочую память и выученные паттерны, и мгновенно делает их доступными в любом другом терминале, IDE или агенте. Состояние сессии, история проекта и накопленные уроки свободно перетекают между вкладками, инструментами и коллегами: без ручной синхронизации, без потери контекста. Вся память живёт в `~/.egc` на вашей машине, зашифрована AES-256-GCM, хранится отдельно для каждой ветки проекта и никогда не коммитится в репозиторий.

### Guardian: Встроенные Ограждения Безопасности

Вторая половина мозга держит ограждения в фоне. Она проверяет команды до выполнения, останавливает рискованные записи, сжимает контекст до переполнения, оркестрирует многошаговые задачи между агентами и учится на каждой поправке, и всё это без единого вызова инструмента с вашей стороны. Невидимая страховочная сеть, которая держит контекст лёгким, действия безопасными, а процессы автономными.

### Token Crusher: Мозг Фильтрует Шум Прежде Чем Запомнить

Мозг не только помнит: он фильтрует. Прежде чем вывод шелла дойдёт до модели, Token Crusher сжимает git-логи, шум тестов, спам установки и гигантские JSON до 90%, сохраняя каждую ошибку и предупреждение. Просто спросите «сколько я сэкономил?» на любом языке, и ответ придёт из вашего локального журнала бесплатно: сессии дешевле, контекст живёт дольше.

## Библиотека Промптов

В качестве бонуса EGC даёт доступ к 63 агентам, 230 скиллам и 77 командам, плюс 111 правил: специалисты, которые сами ревьюят ваш код, руководства по лучшим практикам для каждого языка и ситуации, ярлыки, выполняющие целые последовательности задач, и правила стиля, которые держат код единообразным. Всё написано на основе реальных инженерных сессий, а не теории. Не хотите этим пользоваться? Пожалуйста: постоянная память EGC работает точно так же.

## Быстрый Старт

Второго шага нет. Откройте любой из своих ИИ-инструментов и просто говорите: «привет», «продолжим», «запомни это решение», на любом языке. Сессия регистрируется сама, память загружается сама, и каждая открытая вкладка уже знает, что делают остальные: две вкладки Cursor, терминал Claude Code и сессия Antigravity делят один живой контекст, одновременно.

Живая панель с активностью, токенами и расходами ваших агентов запускается сама сразу после установки. Предпочитаете ручное управление? Все команды описаны в [руководстве по установке](../../docs/installation.md): скорее всего, вам ни одна не понадобится.
